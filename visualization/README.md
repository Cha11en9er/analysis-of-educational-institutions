# Визуализация (Visualization)

Модуль для визуализации данных о школах на карте.

## Структура

- `vl_data/` - папка с данными
  - `vl_input/` - входные данные (JSON файлы)
- `vl_src/` - папка с исходным кодом
  - `vl_main.py` - основной скрипт генерации карты
  - `school_map.html` - HTML файл с интерактивной картой

## Функционал

Модуль создает интерактивную карту с расположением школ на основе геокодированных данных.

## Использование

```bash
python visualization/vl_src/vl_main.py
```

Входной файл: `vl_data/vl_input/vl_test_input_data.json`
Выходной файл: `vl_src/school_map.html` - интерактивная карта
