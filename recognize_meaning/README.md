# Распознавание смысла (Recognize Meaning)

Модуль для анализа отзывов о школах: извлечение тем и определение тональности.

## Структура

- `rm_data/` - папка с данными
  - `rm_input/` - входные данные (JSON файлы с отзывами)
  - `rm_output/` - выходные данные (JSON файлы с проанализированными отзывами)
- `rm_src/` - папка с исходным кодом
  - `rm_main.py` - основной скрипт анализа отзывов
  - `rm_test_main.py` - тестовый скрипт
  - `insert_into_excel.py` - вставка результатов в Excel
  - `ОПИСАНИЕ_СТОЛБЦОВ_ДЛЯ_АНАЛИТИКА.md` - описание столбцов Excel

## Функционал

Модуль обрабатывает отзывы о школах и извлекает:
- **Темы**: ремонт, учителя, еда, администрация, буллинг, инфраструктура, охрана, уборка
- **Тональность**: позитивная (pos), негативная (neg), нейтральная (neu)
- **Общая тональность**: общая оценка отзыва

## Выходной формат

```json
{
  "topics": {
    "учителя": "pos",
    "еда": "neg",
    "ремонт": "pos"
  },
  "overall": "pos"
}
```

## Использование

```bash
python recognize_meaning/rm_src/rm_main.py
```

Входной файл: `rm_data/rm_input/rm_input_data.json`
Выходной файл: `rm_data/rm_output/rm_output_data.json`

## Экспорт в Excel

```bash
python recognize_meaning/rm_src/insert_into_excel.py
```

Результаты сохраняются в Excel файл с детальной статистикой по темам для каждой школы.
