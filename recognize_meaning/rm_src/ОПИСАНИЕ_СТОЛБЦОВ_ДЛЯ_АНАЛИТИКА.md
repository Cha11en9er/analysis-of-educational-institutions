# ОПИСАНИЕ СТОЛБЦОВ В EXCEL (лист 'schools_2_stage')

## БАЗОВАЯ ИНФОРМАЦИЯ О ШКОЛЕ

- **ID** - уникальный идентификатор школы
- **Название на 2ГИС** - полное название школы из 2ГИС
- **Геокоординаты** - координаты местоположения школы
- **Кадастровый номер** - кадастровый номер здания
- **Рейтинг 2ГИС** - рейтинг школы на платформе 2ГИС
- **Рейтинг Яндекс** - рейтинг школы на Яндекс.Картах
- **Ссылка на Яндекс** - прямая ссылка на страницу школы в Яндекс.Картах
- **Ссылка на отзывы Яндекс** - ссылка на отзывы о школе в Яндекс.Картах
- **Ссылка на 2ГИС** - прямая ссылка на страницу школы в 2ГИС
- **Ссылка на отзывы 2ГИС** - ссылка на отзывы о школе в 2ГИС

## ИНФРАСТРУКТУРНЫЕ ПОКАЗАТЕЛИ

- **modernity_score** (0-100) - рейтинг 'свежести здания', учитывает год постройки и реконструкции
- **scale_score** (0-100) - рейтинг масштаба здания (площадь и этажность)
- **amenities_score** (0-70) - рейтинг оснащенности спортивными объектами (бассейн, стадион, ФОК, спортплощадка)
- **Infrastructure_Score** (0-100) - итоговый инфраструктурный скор школы (комплексная оценка)
- **Effective_Year** - эффективный год здания (максимум из года постройки и реконструкции)
- **Age** - возраст здания в годах

## КЛАСТЕРИЗАЦИЯ

- **Cluster** и **Cluster_Name** - к какому кластеру относится школа по схожести инфраструктуры:
  - 0: "Крепкие середняки"
  - 1: "Новые и крупные комплексы"
  - 2: "Старый фонд с базовой инфраструктурой"
  - 3: "Школы с уникальным оснащением (бассейны, стадионы)"

## АНАЛИЗ ОТЗЫВОВ ПО ТЕМАМ

Для каждой темы (Администрация, Буллинг, Еда, Инфраструктура, Охрана, Ремонт, Уборка, Учителя) представлены следующие метрики:

- **<Тема>_Количество_отзывов** - общее количество отзывов по данной теме
- **<Тема>_Количество_позитивных** - количество положительных отзывов по теме
- **<Тема>_Количество_негативных** - количество отрицательных отзывов по теме
- **<Тема>_Доля_негативных_проц** - доля негативных отзывов от общего количества (в процентах, 0-1)
- **<Тема>_Тональность** - общая тональность отзывов по теме (-1 до +1, где -1 полностью негативно, +1 полностью позитивно)

### Примеры столбцов:
- Администрация_Количество_отзывов
- Администрация_Количество_позитивных
- Администрация_Количество_негативных
- Администрация_Доля_негативных_проц
- Администрация_Тональность

Аналогично для всех остальных тем: Буллинг, Еда, Инфраструктура, Охрана, Ремонт, Уборка, Учителя.

---

**Примечание:** Если для какой-то темы нет отзывов, значения могут быть равны 0 или null (пусто).
