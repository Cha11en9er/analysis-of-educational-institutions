# Анализ смысла отзывов с помощью нейронных сетей

Здесь будет код для анализа тональности и смысла спарсенных отзывов с использованием нейронных сетей.

## Планируемые возможности

- Определение тональности (положительный/отрицательный/нейтральный)
- Извлечение ключевых тем и проблем
- Классификация отзывов по категориям
- Оценка важности отзывов
