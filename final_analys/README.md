# Финальный анализ (Final Analysis)

Модуль для финального анализа данных о школах.

## Структура

- `fa_data/` - папка с данными
  - `fa_input/` - входные данные (JSON файлы)
- `fa_src/` - папка с исходным кодом
  - `fa_main.py` - основной скрипт анализа
  - `fa_main_ai.py` - анализ с использованием AI

## Функционал

Модуль выполняет финальный анализ данных о школах, включая:
- Анализ инфраструктуры
- Анализ отзывов
- Кластеризацию школ

## Использование

```bash
python final_analys/fa_src/fa_main.py
python final_analys/fa_src/fa_main_ai.py
```
