# Парсер сайта uchi.ru

Этот скрипт парсит данные школ с сайта uchi.ru, извлекая названия школ из элементов с классом `RatingItem_name__FBvkh`.

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Использование

```bash
python parser.py
```

## Что делает скрипт

1. Парсит страницы с 1 по 15 на сайте uchi.ru
2. Извлекает названия школ из элементов с классом `RatingItem_name__FBvkh`
3. Сохраняет все найденные данные в файл `schools_data.txt`
4. Удаляет дубликаты автоматически

## Настройки

- `start_page` и `end_page` - можно изменить диапазон страниц для парсинга
- `filename` - можно изменить имя выходного файла
- Между запросами добавлена пауза в 1 секунду для избежания блокировки

## Выходной файл

Результаты сохраняются в `schools_data.txt` в формате:
```
Данные школ с сайта uchi.ru
==================================================

1. Название школы 1
2. Название школы 2
...

Всего найдено школ: X
```
