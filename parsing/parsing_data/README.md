# Объединение данных парсинга (Parsing Data)

Модуль для финального объединения данных о школах из всех источников (2ГИС и Яндекс.Карты).

## Структура

- `pd_data/` - папка с данными
  - `pd_output/` - выходные данные (объединенные данные всех школ)
- `pd_src/` - папка с исходным кодом
  - `parsing_data_main.py` - основной скрипт объединения

## Функционал

Скрипт объединяет данные о школах из разных источников:
1. Нормализует названия школ (убирает МОУ/МАОУ и т.д.)
2. Нормализует адреса (убирает "Саратов", индексы и т.д.)
3. Объединяет записи по адресу или названию
4. Создает финальный JSON файл со всеми школами

## Использование

```bash
python parsing/parsing_data/pd_src/parsing_data_main.py
```

Входные файлы (автоматически определяются):
- `parsing/yandex_maps/ym_data/ym_output/ym_full_school_data.json` - данные из Яндекс.Карт
- `parsing/2gis/2gis_data/tgis_output/2gis_all_school_with_info.json` - данные из 2ГИС

Выходной файл:
- `pd_data/pd_output/gold_school_data.json` - финальные объединенные данные
