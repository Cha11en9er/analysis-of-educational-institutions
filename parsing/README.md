# Парсинг данных (Parsing)

Модуль для парсинга данных о школах из различных источников: 2ГИС, Яндекс.Карты, Google Maps, Uchi.ru.

## Структура

Каждая подпапка представляет отдельный процесс парсинга и имеет структуру:
- `*_data/` - папка с данными (input/output)
- `*_src/` - папка с исходным кодом

### Подпапки

- **2gis/** (`tgis_*`) - парсинг школ и отзывов из 2ГИС
- **yandex_maps/** (`ym_*`) - парсинг школ и отзывов из Яндекс.Карт
- **google_maps/** (`gm_*`) - парсинг школ из Google Maps
- **uchi_ru/** (`ur_*`) - парсинг школ с сайта Uchi.ru
- **find_cadastral_number/** (`fcn_*`) - поиск кадастровых номеров
- **vectorization_and_merging/** (`vam_*`) - векторизация и объединение данных из разных источников
- **parsing_data/** (`pd_*`) - финальное объединение данных из всех источников

## Зависимости

- `selenium` - для автоматизации браузера
- `beautifulsoup4` - для парсинга HTML
- `sentence-transformers` - для векторизации текста
- `scikit-learn` - для вычисления схожести

## Использование

Каждый подмодуль имеет свои скрипты для запуска. См. README в каждой подпапке для деталей.
